This code is a supplemental to the scientific paper of Botavara et. al.

The notebooks were restructured and generated using co-pilot through VS-CODE based on the codebase written by the members of the QuICMaPP Facility at the University of the Philippines.
The scope of the code in the notebooks only serve as reference for the evaluation section of the pipeline. You may find the details of the training parameters in the methodology of the article.

Two types of notebooks are provided, one for CPU-only devices and the other for CUDA-enabled devices.
